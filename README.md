## ⚙️ Installation
1. Clone the Repository

```shell
$ git clone https://github.com/BBOXEEEE/braille-education-app-server
$ cd braille-education-app-server
```

2. Install the Package List

```shell
$ pip install -r requirements.txt
```

3. Get start Flask Server

```shell
$ python app.py
```
